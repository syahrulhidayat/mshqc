## 🔍 Verification

Setelah instalasi, verifikasi dengan:

```bash
# Download test script
wget https://raw.githubusercontent.com/syahrulhidayat/mshqc/main/scripts/check_installation.py

# Run diagnostic
python check_installation.py
```
